# FATAH | Gateway

![Gateway Preview](https://img.shields.io/badge/Status-Live-success?style=for-the-badge&color=2563eb)
![Tech Stack](https://img.shields.io/badge/Tech-React%20%7C%20Vite%20%7C%20GSAP-0a0a0a?style=for-the-badge)

A modern, high-performance link portal (Linktree alternative) built for my personal ecosystem. This gateway routes visitors to my portfolio, journal, GitHub, LinkedIn, and direct contact channels. 

Live Demo: [g.fatahmr.my.id](https://g.fatahmr.my.id)

## ✨ Features

- **Blazing Fast Performance:** Optimized with WebP assets, resource preloading, and a highly performant "Solid Translucency" design that runs smoothly at 60 FPS on mobile devices (no heavy `backdrop-filter` GPU usage).
- **GSAP Animations:** Smooth and cinematic entrance animations, including a custom page preloader.
- **Dark/Light Mode:** Seamless theme switching with localized color palettes.
- **Multilingual Support:** Dynamic language switching (Indonesian `ID` and English `EN`) managed via React Context.
- **Mobile-First UX:** Sticky hover states eliminated for touchscreens, ensuring a flawless interactive experience.
- **PWA Ready:** Includes custom favicons and a fully configured Web App Manifest.

## 🛠️ Tech Stack

- **Framework:** React + Vite
- **Styling:** Vanilla CSS (CSS Variables, Flexbox, Media Queries)
- **Animation:** GSAP (GreenSock Animation Platform)
- **Icons:** Lucide React

## 🚀 Getting Started

To run this project locally, make sure you have Node.js installed.

1. Clone the repository:
   ```bash
   git clone https://github.com/fatahilah-mr/gateway.git
   ```
2. Navigate to the project directory:
   ```bash
   cd gateway
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
